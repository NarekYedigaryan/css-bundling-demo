 # CSS Bundling & Transformation Demo

This project demonstrates how modern bundlers transform and bundle CSS with source maps. Multiple SCSS files are combined into a single optimized CSS file while maintaining debuggability.

## Features
- Multiple SCSS modules with imports
- PostCSS processing (autoprefixer, nesting)
- Source maps for development and production
- CSS minification in production
- Live demo of CSS transformation

## Setup & Run

### Prerequisites
- Node.js 16+ and npm

### Installation
```bash
git clone https://github.com/NarekYedigaryan/css-bundling-demo.git
cd css-bundling-demo
